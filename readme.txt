Open YoYo.sln file in VS 2019

make sure installed .Net Core SDK version is 3.1 or later
Make "web" as start up project and and run.

Projects
=========

web => Application Layer (UI).
Yoyo.Models => class library for models and view models.
Yoyo.Service => class library for service implementation.
Yoyo.ServiceContracts => Service contracts



Technologies used
=================

ASP.Net Core 3.1 
MVC
Web Api
Jquery
Bootstrap
